Proyecto Spring: sesiones de usuario

Proyecto "tinder de mascotas", desarrollado en el Curso de Programación
Web FullStack a través de Egg Cooperation y Gobierno de la Provincia de Mendoza.

Esta parte del proyecto, desarrolla la funcionalidad de logeo de 
los usuarios de la página web con su respectivo rol (admin/usuario).

Tecnologías aplicadas: 
- Spring Framework
- Java (maven)
- Java Script
- Html (templates/fragments)
- Css
- Thymeleaf
- JPA Repository
- Sql
